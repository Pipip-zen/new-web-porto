Vibe feature loop:

1. Read only the files needed to identify the fastest insertion point.
2. Ship a thin vertical slice with visible output or a runnable state.
3. Verify the slice quickly with a browser check, screenshot, or command.
4. Tighten the UI and interaction details.
5. Run a final pass for accessibility, responsiveness, and obvious perf issues.

Definition of good first pass:
- User can see it.
- Main interaction path works.
- Code can still be refined later without redoing the whole feature.
