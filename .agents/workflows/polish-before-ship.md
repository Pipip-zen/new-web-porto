Polish before ship:

1. Check visual hierarchy: spacing, type scale, emphasis, and contrast.
2. Check interaction quality: hover, focus, pressed, loading, empty, and error states.
3. Check responsiveness on narrow and wide screens.
4. Check accessibility basics: labels, keyboard reachability, alt text, semantics.
5. Check performance smells: obvious waterfalls, oversized client work, needless rerenders.

Stop when the experience feels intentional, not merely complete.
